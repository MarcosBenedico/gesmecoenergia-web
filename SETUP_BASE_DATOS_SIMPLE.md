# 🚀 SETUP DE BASE DE DATOS - 2 MINUTOS

## ✅ 3 PASOS SOLAMENTE

### Paso 1: Abre Supabase
```
https://app.supabase.com/
```

### Paso 2: SQL Editor → New Query
- Menú izquierdo
- SQL Editor
- Botón "New Query" (abajo)

### Paso 3: Copia y Pega

**Abre el archivo:** `SQL_PARA_PEGAR.sql`

**Copia TODO el contenido** (Ctrl+A, Ctrl+C)

**En Supabase:** Pega en el editor (Ctrl+V)

**Ejecuta:** Botón "RUN" (arriba derecha)

---

## ¡LISTO! ✅

Cuando veas ✅ **Success**, la tabla está creada.

Luego:
```bash
npm run dev
```

Y entra a: http://localhost:3000 → Panel Gestión → Generador Fotovoltaico

---

## 🎯 Si algo falla

Punto 1: ¿Copiaste TODO el SQL?
- ✓ Desde `DROP TABLE` hasta el final

Punto 2: ¿Estás en la página correcta?
- ✓ SQL Editor (no "Query Builder")
- ✓ New Query (no un query existente)

Punto 3: ¿Clickeaste "RUN"?
- ✓ Botón arriba derecha del editor

Si aún falla, mándame el error que dice.
